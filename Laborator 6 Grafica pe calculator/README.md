 Laborator 6 – Blending și Multitexturing

Acest proiect realizează diferite tehnici de blending și multitexturing folosind OpenGL ES în Android Studio.
Aplicația conține mai multe moduri de afișare:
- Alpha blending standard
- Alpha blending cu ordine inversată
- ColorMask
- Culori pe fiecare vârf (per-vertex colors)
- Texture blending
- Multitexturing
Proiectul a fost realizat în Java folosind OpenGL ES și GLSurfaceView.
Pentru schimbarea modurilor se atinge ecranul aplicației.

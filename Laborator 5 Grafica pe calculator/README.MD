Acest proiect demonstrează utilizarea texturilor în OpenGL ES în Android Studio.
Aplicația afișează un pătrat 2D pe care este aplicată o imagine folosind coordonate de textură (s,t).

Funcționalități:
- încărcarea unei texturi din res/drawable
- aplicarea texturii pe un obiect OpenGL
- animația texturii
- mișcarea obiectului pe ecran
- utilizarea GL_TEXTURE_2D și blending

Tehnologii utilizate:
- Java
- Android Studio
- OpenGL ES

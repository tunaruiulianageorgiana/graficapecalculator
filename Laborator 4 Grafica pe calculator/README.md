Acest folder conține:
- Documentația proiectului si printscreen-uri (fișier Word)
- Codul aplicației Android (arhivă .zip).

Aplicația realizează randarea unui cub tridimensional utilizând OpenGL ES pe platforma Android. 
Cubul este animat printr-o mișcare verticală sinusoidală și o rotație continuă în jurul axelor, oferind un efect vizual dinamic. În plus, este implementat un sistem de iluminare de bază, care utilizează normalele suprafețelor pentru a evidenția efectele de lumină asupra obiectului 3D. 
Aplicația demonstrează utilizarea conceptelor fundamentale de grafică pe calculator: transformări geometrice, buffer-e de vertecși, culori și iluminare.
